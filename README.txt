DiscordBOTコンソール(GUI版) Ver1.0.20 ビルド20
[メインアプリケーション機能]
・設定から[!]から始まるコマンド情報が格納される設定ファイルディレクトリの指定
・設定から各フォームの前回終了座標位置を取得するフラグ
・設定からユーザが入力したコマンドを削除するかどうかのフラグ
・編集からオリジナルコマンドの登録(メッセージ形式、Embed形式)
・プラグインから追加機能に基づいた設定が可能(入退出ログ、アンケートQA、天気予報)

[DiscordBot.Plugin.UserJoining_Leaving.dll]
サーバへの入退室ログ機能!!

[DiscordBot.Plugin.QA.dll]
リアクション操作メソッドが非同期的に実行されます!!

[DiscordBot.Plugin.Weather.dll]
天気予報WebサーバからAPI通信でリアルタイムの天気情報を確認できます!!


[DiscordBot.Plugin.Kick.dll]
サーバからユーザを指定してキックする機能!!

[DiscordBot.Plugin.Delete.dll]
サーバのチャンネル内のあらゆるメッセージを一括で削除する機能!!
